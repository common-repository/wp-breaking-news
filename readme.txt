== Description ==

Puts a marquee on the top of your wordpress blog showing messages you want.

== Installation ==

1. Download the zip file and extract the contents.
2. Upload the folder to your plugins directory (wp-content/plugins/).
3. Activate the plugin through the 'plugins' page in WP.
4. After yo activate the plug in, you will see an new option called "WP Breakeing News".

== Adding message ==
5. Go to the "WP Breaking News" option in the Settings menu.
6. Click in the "Add New Message"
7. Done ! The messages will be showed on the marquee on the screen screen
